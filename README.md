# Password Strength Analyzer

This project analyzes password strength using Java and Python.

## 🚀 Java Version (Main)
- Built using Java Swing (GUI)
- Displays password strength
- Shows suggestions
- Includes color strength bar
- Estimates crack time

## 🐍 Python Version
- Initial prototype using Tkinter

## 🔧 Features
- Regex validation
- Strength scoring
- Crack time estimation
- Common password detection

## ▶️ How to Run
javac PasswordCheckerGUI.java
java PasswordCheckerGUI
